# Module 2 - Data Structures and Algorithms

Part of Cognizant Digital Nurture 5.0 (DN 5.0) Deep Skilling Program — DotNet FSE track.

## Contents

| Folder | Topic | Files |
|---|---|---|
| `Arrays/` | Array traversal, insertion, deletion, search, reverse | `ArrayOperations.cs` |
| `LinkedList/` | Singly, Circular, Doubly linked list operations | `SinglyLinkedList.cs`, `CircularLinkedList.cs`, `DoublyLinkedList.cs` |
| `Sorting/` | Bubble, Insertion, Heap, Quick, Merge sort | `SortingAlgorithms.cs` |
| `Searching/` | Linear search, Binary search (iterative + recursive) | `SearchingAlgorithms.cs` |

## How to Run

Each `.cs` file is a standalone console program with its own `Main` method.

1. Open the file in Visual Studio / VS Code.
2. Create a new Console App (.NET) project.
3. Replace `Program.cs` content with the desired file's content (or add the file to the project and rename its `Main` method if running multiple files in one project).
4. Run with `dotnet run` or press F5.

## Topics Covered

- **Arrays**: representation in memory, traversal, time complexity of operations
- **Linked Lists**: single, circular, and double linked list — insert, delete, search, traverse
- **Sorting**: worst/average/best-case analysis of Bubble, Insertion, Heap, Quick, and Merge sort
- **Searching**: Linear Search (O(n)) and Binary Search (O(log n)) on sorted arrays

## Time Complexity Summary

| Algorithm | Best | Average | Worst |
|---|---|---|---|
| Bubble Sort | O(n) | O(n²) | O(n²) |
| Insertion Sort | O(n) | O(n²) | O(n²) |
| Heap Sort | O(n log n) | O(n log n) | O(n log n) |
| Quick Sort | O(n log n) | O(n log n) | O(n²) |
| Merge Sort | O(n log n) | O(n log n) | O(n log n) |
| Linear Search | O(1) | O(n) | O(n) |
| Binary Search | O(1) | O(log n) | O(log n) |
